# Social Backend API

"클라우드 커뮤니티" 서비스의 백엔드 API 서버를 구현합니다.

지난주에 각자 작성한 **REST API Docs**를 기반으로 실제 동작하는 서버를 개발합니다.

수행 방법에 대한 상세는 노션 페이지 내 과제 제출 페이지 (4주차 과제) 에서 확인해주세요!
